# Imtihan
Imtihan is a platform for teachers to make exams and share them!

Imtihan was made using Antigravity IDE using gemini 3 Pro (for the ui) and gemini 3 Flash (for the backend)

I only remember the first prompt "create me a website which has 2 sections one is the editor and the other a community section, once you click on the editor you are met with a word-like paper on your left and on your right a control ui that consists of the aspects of a jordanian school exam paper (title, logo, mcq questions, fill in the blanks, marks and a question editor with an export button at the top left into pdf and an option to share it to the community and add a database system (until i figure out a db server later)"
